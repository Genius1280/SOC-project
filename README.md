
# SOC Incident Investigation

Учебный проект по расследованию инцидента информационной безопасности.

## Выполненные задачи

- Анализ инцидента в IRP-системе TheHive
- Определение типа инцидента
- Построение цепочки событий
- Анализ сетевой и endpoint-активности
- MITRE ATT&CK mapping
- Подготовка рекомендаций по реагированию
- Документирование результатов расследования

## Используемые инструменты

- TheHive
- Wazuh
- Suricata
- ELK Stack
- Linux
- MITRE ATT&CK
